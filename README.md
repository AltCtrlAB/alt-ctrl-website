# Alt Ctrl — Website

Landing page for Alt Ctrl, built with React 19 + TypeScript + Tailwind CSS v4 (frontend) and FastAPI (backend).

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React 19, TypeScript, Tailwind CSS v4, Motion, shadcn/ui |
| Backend | Python 3.12+, FastAPI, Uvicorn |
| Package managers | pnpm (Node), uv (Python) |
| Build tool | Vite 6 |
| Animations | Motion (Framer Motion) |
| Icons | Lucide React |
| UI components | shadcn/ui (New York style) + 21st.dev |

---

## Prerequisites

- [Node.js](https://nodejs.org/) 20+
- [pnpm](https://pnpm.io/) — `npm install -g pnpm`
- [Python](https://python.org/) 3.12+
- [uv](https://docs.astral.sh/uv/) — `pip install uv` or `curl -LsSf https://astral.sh/uv/install.sh | sh`
- `make` (Git Bash on Windows, or `brew install make` on macOS)

---

## Setup

```bash
# Install all dependencies (frontend + backend)
make install

# Copy env file and fill in values
cp backend/.env.example backend/.env
```

---

## Development

Run the backend and frontend in two separate terminals:

```bash
# Terminal 1 — FastAPI on http://localhost:8000
make dev-backend

# Terminal 2 — Vite on http://localhost:5173
make dev-frontend
```

The Vite dev server proxies `/api/*` requests to the FastAPI backend automatically.

---

## Production Build

```bash
# Build React → backend/static/ (FastAPI serves it)
make build

# Start production server
make start
```

## Docker

Requires [Docker Desktop](https://www.docker.com/products/docker-desktop/).

```bash
# 1. Create your env file
cp backend/.env.example backend/.env
# Edit CONTACT_EMAIL, CORS_ORIGINS if needed

# 2. Build image and start
docker compose up --build
# → http://localhost:8000
```

The multi-stage image builds the React app in a Node 22 container, then copies the output into a
Python 3.12-slim runtime that serves both the API and the SPA from a single process on port 8000.
All env vars from `backend/.env` are injected at runtime — no rebuild needed when changing config.

## Adding shadcn/ui Components

```bash
cd frontend
npx shadcn@latest add button
npx shadcn@latest add card input textarea badge

# From 21st.dev
npx shadcn@latest add "https://21st.dev/r/<author>/<component>"
```

## Pre-launch Checklist

Before going live, complete the following:

- [ ] Replace placeholder case studies in `frontend/src/lib/company.ts` with real client data
- [ ] Create an `og:image` (1200×630 PNG) and add its URL to the `<meta>` tags in `frontend/index.html`
- [ ] Wire up real email delivery in `backend/main.py` (aiosmtplib, Resend, or SendGrid)
- [ ] Set `CORS_ORIGINS` in `backend/.env` to your production domain (e.g. `https://alltunderkontroll.se`)
- [ ] Add analytics (Plausible, GA4, or similar) to `frontend/index.html`
- [ ] Add `robots.txt` sitemap URL once DNS is live (already at `frontend/public/robots.txt`)
- [ ] Point your domain DNS to the server and configure TLS (Caddy / nginx / Fly.io / Railway)
